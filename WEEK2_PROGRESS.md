#### ✨ التقارير الجديدة المضافة:

##### 📅 **1. تقرير التدقيق اليومي** (`get_daily_audit_report`)
تقرير شامل لجميع عمليات اليوم يشمل:

**أ) إحصائيات الزيارات:**
- التوزيع حسب الحالة (OPEN, IN_PROGRESS, COMPLETED, ARCHIVED)
- التوزيع حسب النوع (REGULAR, FOLLOW_UP, CONSULTATION, EMERGENCY)
- إجمالي الزيارات

**ب) إحصائيات الدفع:**
- المبالغ المحصلة حسب طريقة الدفع (نقدي، بطاقة، تأمين)
- المبالغ المحصلة حسب الحالة (مؤكد، معلق، ملغي)
- إجمالي المعاملات المالية

**ج) تحليل الدفع القسري:**
- عدد الدفعات القسرية
- المعتمدة vs المعلقة
- النسبة المئوية من إجمالي الزيارات
- قائمة تفصيلية بكل حالة (السبب، الموافق، التاريخ)

**د) تحليل التأمين:**
- عدد زيارات التأمين
- إجمالي المبلغ المفوتر للتأمين
- حصة المريض (المطلوبة vs المحصلة)
- التوزيع حسب مزود التأمين

**ه) قضايا التدقيق (Audit Issues):**
1. **زيارات بدون دفع** (Severity: MEDIUM)
2. **دفع قسري بدون موافقة** (Severity: HIGH)
3. **دفعات ملغاة** (Severity: MEDIUM)
4. **دفعات نقدية كبيرة** > 1000 شيكل (Severity: LOW)

**مثال على التقرير:**
```json
{
  "success": true,
  "date": "2025-10-12",
  "visits": {
    "total": 45,
    "by_status": {"COMPLETED": 30, "IN_PROGRESS": 10, ...},
    "by_type": {"REGULAR": 35, "EMERGENCY": 5, ...}
  },
  "payments": {
    "total_collected": 15750.00,
    "by_method": {
      "CASH": {"count": 25, "amount": 8500},
      "CARD": {"count": 15, "amount": 5250},
      "INSURANCE": {"count": 5, "amount": 2000}
    }
  },
  "force_payments": {
    "total": 2,
    "approved": 1,
    "pending": 1,
    "percentage": 4.44
  },
  "insurance": {
    "total_visits": 8,
    "patient_share_collected": 1200,
    "patient_share_pending": 300
  },
  "audit_issues": [
    {
      "type": "FORCE_PAYMENT_PENDING",
      "severity": "HIGH",
      "count": 1,
      "message": "1 دفع قسري بانتظار الموافقة"
    }
  ]
}
```

**ب) الإحصائيات المالية:**
- إجمالي الإيرادات
- التوزيع حسب طريقة الدفع (نقدي، بطاقة، تأمين)
- الإيرادات اليومية (daily_revenue)
- متوسط قيمة المعاملة
- متوسط الإيرادات اليومي

- العدد الإجمالي
- المعتمد vs المرفوض vs المعلق
- النسبة من إجمالي الزيارات
- معدل الموافقة (approval_rate)
- إجمالي المبالغ والمحصل

**د) تحليل التأمين:**
- عدد الزيارات
- النسبة من إجمالي الزيارات
- إجمالي المفوتر للتأمين
- حصة المريض (المطلوبة vs المحصلة vs المعلقة)
- معدل التحصيل (collection_rate)
