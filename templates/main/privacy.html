سياسة الخصوصية

آخر تحديث: 2025-12-17

1. مقدمة

نحن في {{ branding.organization_name if branding else 'المركز الصحي المتخصص' }} نولي اهتماماً كبيراً لخصوصية بياناتكم. توضح هذه السياسة كيفية جمعنا واستخدامنا وحمايتنا لمعلوماتك الشخصية.

2. جمع المعلومات

نقوم بجمع المعلومات التي تقدمها لنا مباشرة عند استخدامك للنظام، مثل:

  • معلومات الحساب (الاسم، البريد الإلكتروني، رقم الهاتف).
  • البيانات الطبية والصحية اللازمة لتقديم الخدمات.
  • سجلات الدخول والاستخدام لتحسين الأداء والأمان.

3. استخدام المعلومات

نستخدم المعلومات التي نجمعها للأغراض التالية:

  • تقديم الخدمات الطبية وإدارتها.
  • تحسين تجربة المستخدم وتطوير النظام.
  • التواصل معك بخصوص المواعيد والتقارير الطبية.
  • ضمان أمن وسلامة النظام.

4. حماية البيانات

نحن نلتزم بتطبيق معايير أمان عالية لحماية بياناتك من الوصول غير المصرح به أو التغيير أو الإفشاء أو الإتلاف. نستخدم تقنيات التشفير وإجراءات الأمان المادية والإلكترونية المناسبة.

5. مشاركة المعلومات

لا نقوم ببيع أو تأجير بياناتك الشخصية لأطراف ثالثة. قد نشارك المعلومات فقط في الحالات التالية:

  • مع مقدمي الخدمات الطبية المعتمدين (مثل المختبرات أو الصيدليات) لغرض العلاج.
  • عندما يكون ذلك مطلوباً بموجب القانون.

6. حقوقك

لديك الحق في الوصول إلى بياناتك الشخصية وتصحيحها أو طلب حذفها (مع مراعاة المتطلبات القانونية للاحتفاظ بالسجلات الطبية).
